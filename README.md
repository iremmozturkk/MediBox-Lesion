# 🧠 3D Lung Lesion Segmentation

This project focuses on automatic detection and segmentation of lung lesions in 3D CT images using deep learning methods.

---

## 📌 Project Overview

The aim of this project is to process 3D CT scans and generate segmentation masks for lesion regions using a 3D U-Net based architecture.

The project is based on the **MELA2022 dataset**.

---

## ⚙️ Features

- 3D CT image processing  
- Lesion segmentation (3D U-Net)  
- Model performance evaluation  
- Visualization of results  
- Web-based interface (Streamlit)  

---

## 🧪 Evaluation Metrics

- Dice Score  
- IoU  
- Precision  
- Recall
